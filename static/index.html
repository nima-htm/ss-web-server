Welcome to SS !

A lightweight alternative to nginx for containerized applications

This page is being served by the Go web server with reverse proxy capabilities. This lightweight server can replace nginx in containerized environments.

Reverse Proxy

Forwards requests to backend services like FastAPI or Django applications

Load Balancing

Distributes requests across multiple backend instances using round-robin algorithm

Static Serving

Efficiently serves static files with proper caching headers

Configuration

This server is configured using a YAML file (default.conf). API requests to /api/ are forwarded to backend servers as defined in the configuration.

Current configuration file: default.conf

API Integration

Requests to /api/* paths are automatically forwarded to backend services (e.g., your FastAPI or Flask application) while other requests serve static files from the ./static directory.
